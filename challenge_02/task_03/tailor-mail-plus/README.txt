== Description:

Tailor Mail Plus is a WordPress plugin that allows you to create customizable contact forms with code-defined fields.
Messages are sent via wp_mail or SendGrid API according to your choices.
Submitted entries are saved in the database in a custom table providing you with a record of all inquiries.
The plugin utilizes shortcodes for easy integration into your website's pages and posts.

This plugin is a solution to a Senior Challenge proposed by [WP Challenges](https://www.wpchallenges.com).

== Installation:

    Upload the plugin folder to the /wp-content/plugins/ directory, or install it directly from the WordPress admin area.
    Activate the plugin through the 'Plugins' menu in WordPress.
